# Admin Dashboard
In this project I am gonna build a web page based on a dashboard design given by the [The Odin Project](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-admin-dashboard), this with the objective of apply the new knowledge on CSS Grid.

## [LIVE PREVIEW](https://bast1ann.github.io/admin-dashboard/)

## Key Objectives
* Apply CSS Grid for the main design.
* Use CSS Custom Properties to organize the CSS code and the webpage's color scheme.
* Add a Theme Switch Button to change the webpage's color scheme.
